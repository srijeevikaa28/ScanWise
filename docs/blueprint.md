# **App Name**: ScanWise

## Core Features:

- Live QR Code Scanning: Uses the device's camera to scan QR codes in real-time, extracting the data encoded within them. Leverages tools to extract specific, formatted data such as names, ingredients, and expiration dates.
- Image-Based QR Code Scanning: Allows users to upload images containing QR codes for data extraction. Supports various image formats and automatically detects and decodes QR codes present in the image.
- Data Storage: Stores scanned product information, including name, ingredients, manufacture date, expiry date, and status, in a MySQL database for future access and management.
- Product Management: Enables users to view and manage scanned product details, update the product status, and adjust quantities.
- Quantity Adjustment: Provides a feature to easily update or insert quantities of scanned items in the inventory.

## Style Guidelines:

- Primary color: Subtle sky blue (#6FB0D3), creating a clean and trustworthy ambiance that highlights scanning accuracy.
- Background color: Very light blue (#F0F8FF), providing a calm and unobtrusive backdrop.
- Accent color: Warm coral (#F08080), used for highlighting actionable items and critical information to guide user interaction.
- Font pairing: 'Poppins' (sans-serif) for headlines, and 'PT Sans' (sans-serif) for body text, blending modernity with readability.
- Use clear and simple icons that represent the functionality of each button or interactive element to improve usability.
- Design a responsive layout that adapts seamlessly to different screen sizes to provide a consistent and intuitive experience across all devices.
- Incorporate smooth, subtle animations for transitions and user feedback, enhancing the app's interactivity without being distracting.
